# Contributing to awesome-openclaw-hardware-projects

Thank you for your interest in contributing!

## Guidelines

### What to Submit

- **Hardware projects** that run OpenClaw or integrate with it
- **Skills** related to hardware control (sensors, motors, cameras, etc.)
- **Tutorials** for setting up hardware with OpenClaw
- **Benchmarks** and performance comparisons

### Requirements

1. **Verify your submission** — Only submit projects you've tested and confirmed work
2. **Include description** — Clear explanation of what the project does
3. **Categorize properly** — Place in the most relevant category
4. **Link to official sources** — Docs, GitHub repos, etc.

### Format

For adding a project, include:
- Project name
- Link to repository/documentation
- Brief description (1-2 sentences)
- Hardware requirements (if applicable)

### Security Note

⚠️ When integrating hardware with AI agents:
- Review skill source code before installation
- Avoid hardcoding credentials or API keys
- Understand what physical actions the agent can trigger
- Consider safety implications of automation (especially for robotics, power control, etc.)

---

## How to Submit

1. Fork this repository
2. Add your project to the appropriate category
3. Submit a Pull Request

Questions? Open an issue first to discuss.
